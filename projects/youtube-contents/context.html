YouTube 콘텐츠 프로젝트 — 배경 컨텍스트

업무 히스토리 (2024년 말 ~)

회사 커머스 서비스 변천

  1. 임프몰 (Imp몰) — 기존 캐릭터 IP 굿즈 편집샵 (캐릭터 IP의 "29cm" 컨셉, 감도 높은 큐레이션). 결과적으로는 성과 미달.
  2. 라인프렌즈 스퀘어 (Square) 통합몰로 컨버팅 — 임프몰 웹 서비스를 베이스로 재구축. - 기존 국내 채널: 스마트스토어 - 기존 해외 채널: 라인프렌즈 컬렉션 (Shopify) - → 국내·해외 모두 커버하는 라인프렌즈 스퀘어 단일 통합몰로 합쳐짐

인스타그램 운영권 인수

  • 임프몰이 운영하던 인스타그램 계정이 통합 컨버팅 후 붕 뜸
  • 그 계정을 인수해 square_b.side (B-side) 계정으로 운영 시작
  • 본 계정(A side)에서 못 다루는 B급 / 매체 지향적 톤의 콘텐츠 제작
  • 주력 포맷: 릴스(Reels)

성과 → 확장

  • 사내 반응이 좋았음
  • B-side에서 검증된 콘텐츠를 A 계정에도 게재 시작
  • YouTube 채널으로도 확장 (현재 콘텐츠 고민 중인 단계)

현재 상황 & 문제 정의

제약

  • 팀 케파 부족: 롱폼 콘텐츠 기획/편집할 물리적 시간이 없음
  • 숏폼(릴스) 한계: 들이는 품 대비 성장 효율 낮음 — 숏폼만으로는 채널 성장 한계

최근 변곡점

  • 므메미무 팝업 홍보 영상 (롱폼 팝업 리뷰)을 직접 제작 → 의외로 터짐 + 구독자 유입
  • 위에서 "유튜브 해봐~" 지시 내려옴

결론

  • 롱폼 제작이 필수가 된 상황
  • 하지만 케파는 그대로 → "최소 리소스로 롱폼을 효율적으로 찍어내는 운영 모델" 이 핵심 과제

변수 실측

케파

  • 촬영 + 편집 + 기획 풀 사이클: 월 1건이 한계
  • 므메미무 영상이 가능했던 이유: 사용자가 기획 방향만 인턴에게 다운로드 → 인턴이 편집/구성 전담 → 사용자는 "빼야 할 부분"만 체크

콘텐츠-시장 적합성 단서

  • 임프몰 시절 캐릭터 IP 이야기: 100~200뷰 (지루함)
  • 므메미무(아이돌 IP) 영상: 1만뷰 (터짐)
  • → "아이돌 팬덤이 궁금해하는 이야기" 가 채널 강점

1차 아이디어: 팟캐스트 포맷 (사용자 제안)

컨셉

  • 주제: 아이돌 IP / 굿즈 / 팬 문화에 대한 실무자 인터뷰 + 회사 내부 토크
  • 포맷: 팟캐스트 (롱폼)
  • 고정 호스트: 본인
  • 회차별 게스트: MD / 마케터 / 실제 팬 로테이션

효율 핵심 (제작 워크플로우)

  1. 인터뷰는 폰으로 그냥 녹음 (촬영 X, 음성 보정 X)
  2. AI 보이스로 음성 재구성 → 팟캐스트화
  3. 화면은 AI 캐릭터들이 쇼파에서 팟캐스트하는 비주얼
  4. 한 번 시스템 구축 후 → "사람 + 주제"만 픽하면 굴러감

파일럿 1편 — 확정 사양

음성 전부 AI 보이스 (raw 녹음 품질 불가)
캐릭터 IP 오리지널 캐릭터 (라인프렌즈 IP 미사용)
호스트 캐릭터 고정 1체 (시리즈 내내 동일)

길이 MAX 20분
