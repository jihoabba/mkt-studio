# 4월 이후 광고 소재 효율 하락 — 진단 & 전략 보완

> 대행사 YTD 리포트 보완 자료. 5월 먼슬리 미팅용.

---

## 1. 진단 — 왜 효율이 떨어지나

### 표면 신호
- **CPC**: 1월 272원 → 4월 327원 → 5월 375원 (꾸준한 상승)
- **CVR**: 3.34% → 2.83% → 2.07% (꾸준한 하락)
- 두 지표가 **동시에 악화** = "더 비싼 클릭이 더 안 사는" 구조
- 특히 **메타 리타겟팅** 효율이 스스·스퀘어 두 스토어 모두 동반 하락

### 진짜 원인 — 비즈니스 구조 문제

**가설: "팝업 위주 운영 + 신제품 공백 + 동일 광고 세팅 유지" 가 만든 복합 결과**

| 단계 | 평소 | 4월 이후 |
|---|---|---|
| 1. 신규 유입 | 신제품·티징 콘텐츠가 신규 트래픽 유입 | 새 콘텐츠 부재 → 신규 유입 감소 |
| 2. 리타겟팅 풀 | 새 방문자가 유입되며 풀 갱신 | 풀이 노후화 (이미 둘러본 사람 = 살 사람은 샀고 안 살 사람은 안 삼) |
| 3. 소재 | 신제품·신상 캠페인 소재로 교체 | 같은 소재 반복 → 광고 피로도 ↑ |
| 4. 결과 | CVR 유지·CPC 안정 | CVR ↓·CPC ↑ (메타 알고리즘이 효율 낮은 풀에 비싼 입찰) |

→ **광고 운영의 문제가 아니라 콘텐츠·상품 사이클의 문제가 광고에 나타나는 것**

### 보조 지표로 확인
- **메타 협력광고**: 광고비 17% / 클릭 21%(1위) / 구매 5.8% / 매출 4.0%
  - 클릭은 잘 받는데 전환이 안 됨 = 소재가 호기심은 자극하지만 구매 의도 전환 실패
- **프렌즈위크 ROAS 63%**: 프로모션 자체로는 매출 부양 효과 약함
- **상시 SA ROAS 1,208% 유지**: 검색 수요(브랜드 자산)는 살아있음 → 문제는 **푸시 채널의 신선도**

## 2. 대응 전략 — 4가지 옵션 + 평가

### 옵션 A. **광고비 효율화 + 프로모션 집중 (수비형)**
- 메타 리타겟팅·DA 광고비 30~40% 축소
- 절감분 → 상시 SA + 시즌 프로모션 (어린이날·여름 등) 집중
- **장점**: 즉시 ROAS 회복, 위험 최소
- **단점**: 볼륨 축소 → 신규 고객 풀 더 빠르게 마름 (악순환 가능)
- 적합 시점: 신제품 라인업이 6~8월에 다시 들어올 게 확실할 때

### 옵션 B. **신규 시장·타겟 발굴 (공격형)**
- 룩어라이크 오디언스 확장 (기존 구매자 기반 1%→3% 확대)
- 신규 지역·연령대 테스트 (일본·동남아 / 30대 후반·40대 IP 향수층)
- 인플루언서 협업으로 새 유입 채널 만들기
- **장점**: 리타겟팅 풀 자체를 키움 → 근본 처방
- **단점**: 초기 ROAS 더 낮을 수 있음, 검증 기간 필요
- 적합 시점: 단기 ROAS 압박이 적고 중장기 성장 KPI 있을 때

### 옵션 C. **콘텐츠·소재 강화 (콘텐츠형)** ⭐ 추천
- "이달의 발굴템" / "직원 픽 베스트" / "캐릭터별 사연 시리즈" 같은 **상시 콘텐츠 포맷**
- 신제품 없어도 신선도를 만드는 큐레이션·기획 콘텐츠
- 팬덤 UGC 큐레이션 (square_b.side 톤 활용)
- **장점**: 신제품 사이클과 무관하게 신선도 유지 가능, 우리 팀 강점과 일치
- **단점**: 콘텐츠 제작 케파 필요
- 적합 시점: **지금 (즉시 시작 가능)**

### 옵션 D. **채널 다각화 (분산형)**
- 메타 협력광고 → 29CM·올웨이즈·CJ온스타일 등 신규 협력광고
- PC 인벤토리 확대 (ROAS 3,236%인데 광고비 15%만 집행 중)
- TikTok·Pinterest 등 미사용 매체 소규모 테스트
- **장점**: 보고서 자체에서도 ZOA × 29CM 사례로 검증됨 (ROAS 849%)
- **단점**: 채널별 학습 기간 필요
- 적합 시점: 지금 (실험 단위로)

## 3. 추천 — C + D + A 일부 조합

### 우선순위 (즉시 ~ 1개월)
1. **콘텐츠 신선도 회복 (C)** — 가장 빠르고 우리 통제 가능
   - 6월부터 "이달의 발굴템" 시리즈 정기화 (월 4편)
   - 기존 SKU 재해석 콘텐츠 = 신제품 효과
   
2. **PC 인벤토리 확대 (D 일부)** — 보고서 데이터로 즉시 정당화 가능
   - PC ROAS 3,236% → SA 광고비 중 PC 비중 15% → 25%로 단계 확대
   
3. **메타 협력광고 효율화 (A 일부)** — 가장 약한 채널부터 정리
   - 신규 소재·카탈로그·오디언스 점검
   - 개선 안 되면 광고비 20% 축소 → 29CM 협력광고 등으로 이전

### 중기 (1~3개월)
4. **29CM × 타 IP 협력광고 실험 (D)** — ZOA 사례 재현 검증
   - BT21, 조구만, ZOA 외 2~3개 IP × 29CM 조합 테스트
   
5. **룩어라이크 신규 타겟 테스트 (B)** — 풀 갱신

### 장기 (분기 단위)
6. **신제품 사이클 자체 개편 논의** — MD·기획팀과
   - 팝업 사이클과 별개로 **상시 라인 신상** 출시 주기 확보 필요성 제기
   - 광고가 살리는 게 아니라 **상품이 살리는 구조** 복원

## 4. 미팅에서 짚을 핵심 메시지 3가지

### 1️⃣ "광고 효율 하락은 광고 운영 문제가 아니라 콘텐츠·상품 공백 문제"
- 데이터로 입증: 상시 SA ROAS 1,208% 유지 (브랜드 자산은 멀쩡)
- 문제는 푸시 채널(메타 리타겟팅·DA)의 신선도 고갈
- → 광고팀이 더 잘한다고 풀리는 문제 아님

### 2️⃣ "단기 처방 = 콘텐츠 신선도, 중기 처방 = 채널 다각화, 장기 처방 = 신상 사이클"
- 즉시 가능한 콘텐츠 강화부터 시작
- 데이터 근거 있는 채널 확장 (PC·29CM)
- 구조 개편은 MD·기획팀과 별도 논의

### 3️⃣ "광고비 무작정 축소는 악순환 우려"
- DA·리타겟팅 컷팅 → 신규 유입 더 마름 → 풀 더 노후화 → 더 떨어짐
- 콘텐츠·채널 다각화로 풀을 다시 채우면서 효율화 병행
