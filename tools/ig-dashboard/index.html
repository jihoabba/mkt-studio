CSV URL 발급 방법
1. Google Sheets 열기 → 파일 → 공유 → 웹에 게시
2. 시트 선택 (콘텐츠 또는 팔로워 탭) → 형식 CSV → 게시
3. 나오는 URL 복사 → 위 칸에 붙여넣기
4. 시트별로 URL이 다릅니다 (gid 값 다름)
